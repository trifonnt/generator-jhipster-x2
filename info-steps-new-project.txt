= Generate SSH key
+01) Generate SSH key
$ ssh-keygen -m PEM -t rsa -b 4096 -C "FMI-MyBudget"
Generating public/private rsa key pair.
Enter file in which to save the key (/home/vagrant/.ssh/id_rsa): /home/vagrant/.ssh/id_rsa-fmi-my-budget

+02) Attach public key to the GitHub repository keys

+02.2) Configure local git to use proper SSH Key
$ git config --add --local core.sshCommand 'ssh -i /home/vagrant/.ssh/id_rsa-fmi-my-budget'
